Beecpetor Assignment

1. Create a free Beeceptor account.
2. Explore the HTTP Callout feature.
3. Read the documentation and understand how it works. https://lnkd.in/dHvBe6Ru
4. Write a #Playwright automation (Javascript) that completes the entire workflow end-to-end. The automation should open Beeceptor, create (or reuse) an endpoint, create and configure an HTTP Callout rule, trigger the the APi call, verify that the HTTP Callout executes successfully with the expected behavior, and clean up any test data if appropriate.

6. Record a 2–3 minute demo video (camera on). Keep it concise and focus on the things that create the most value: your understanding of the feature, your testing strategy, key implementation decisions, and a quick demo of the automation run. If needed, edit the video to keep it under 3 minutes.
7. Submit your Playwright project as a GitHub repository and your demo video
